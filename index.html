Finalizei o curso intensivo de desenvolvimento web fullstack pela escola de programação Digital StartSE, onde apronfundei meu conhecimento em Javascript ultilizando NodeJS, Express, ReactJS, GIT, Sequelize, MySQL, SQL e também melhorando minhas Skills em HTML e CSS e Metodologias Ágeis.
